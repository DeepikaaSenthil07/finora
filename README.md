# 💰 Finora Personal Finance Tracker

A full stack personal finance web application built with React, Node.js, Express, and MongoDB.

## 🌐 Live Demo
**[finora-drab.vercel.app](https://finora-drab.vercel.app/)**

---

## ✨ Features
- 🔐 JWT Authentication : secure signup, login, and session management
- 📊 Dashboard : real-time balance, income/expense charts, recent transactions
- 💸 Transactions : add, filter, search, and delete income/expense entries
- 🎯 Budget : set monthly limits per category with live progress tracking
- 🎉 Onboarding : guided setup flow for new users
- 👤 Profile : update name, email, change password, delete account
- 📱 Fully responsive across mobile, tablet, and desktop

## 🛠️ Tech Stack
| Layer | Technology |
|---|---|
| Frontend | React 18, Vite, React Router DOM v6, CSS Modules |
| Charts | Recharts |
| Backend | Node.js, Express.js |
| Database | MongoDB Atlas, Mongoose |
| Auth | JWT, bcryptjs |
| Deployment | Vercel (frontend), Render (backend) |
